JavaScript30 Challenge

A collection of 30 projects built with vanilla JavaScript — no frameworks, no libraries, just pure fundamentals.

📌 About
This repository contains my solutions to the JavaScript30 challenge by Wes Bos.
The goal is to strengthen core JavaScript skills by building real-world projects using only:
HTML5
CSS3
Vanilla JavaScript (ES6+)

🧠 What I am Learning in this challenge
. DOM manipulation & traversal
. Event handling & delegation
. CSS + JavaScript interactions
. ES6+ features (arrow functions, destructuring, etc.)
. Working with browser APIs
. Animations & transitions
. Debugging and problem solving

JavaScript (ES6+)
